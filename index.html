Introduction

Orcfax is a decentralized oracle service that publishes facts about real world events to the Cardano blockchain.

Getting Started​

Learn how your Cardano smart contracts and scripts can consume data from Orcfax.

Using Orcfax's unique auditing and archiving features, you can verify data published by this oracle.

Orcfax's native FACT token is used by Consumers to pay for oracle publication as well by Validators for staking and rewards.
